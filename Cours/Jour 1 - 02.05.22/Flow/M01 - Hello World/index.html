Au cours de cette formation nous allons avoir 2 matiéres

Lintégration

Le HTML

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, etc

Le développement

Le javascript :

Javascript est un langague de programmation de script employé pour rendre une page web interactive, les comportement de la page et son apparence pourra varier en fonction de certains paramètres.

Le PHP :

Le PHP est un langage de programmation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certaines information (heure, nom de l'utilisateur, formulaire, rempli par l'utilisation etc.

MySQL :

MySQL est le systéme de gestion de basses de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de création de compte) pour ensuite être ré-utilisé au moment appropié (ex: connexion à un site internet avec identifiant et mor de passe).
